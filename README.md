# Ayoub-regd.github.io

Portfolio statique publie sur GitHub Pages.

## Structure publiee

- `index.html`
- `styles.css`
- `app.js`
- `robots.txt`
- `sitemap.xml`
- `404.html`
- `assets/og-image.png`
- `projects/*.html`

## Contraintes de publication

- Aucun fichier AWS de deploiement n'est publie.
- Aucun export CV annexe n'est publie.
- Le PDF CV final doit etre regenere apres mise a jour manuelle du lien portfolio dans `https://showcase.doyoubuzz.com/en/login`.

## Mise a jour du CV

1. Publier le site GitHub Pages.
2. Recuperer l'URL finale `https://ayoub-regd.github.io/`.
3. Mettre a jour le lien portfolio dans Doyoubuzz.
4. Reexporter le PDF.
5. Ajouter le PDF dans `assets/` et reactiver le bouton CV.
